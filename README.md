# nd-upload

[![spm version](http://spmjs.io/badge/nd-upload)](http://spmjs.io/package/nd-upload)

> 基于 WebUploader 的上传组件

## 安装

```
$ spm install nd-upload --save
```

## 使用

```js
var Uploader = require('nd-upload');

使用时，需手动配置vendor下的Uploader.swf
new Uploader({
	swf: ''//swf地址
})
// use nd-upload
```
